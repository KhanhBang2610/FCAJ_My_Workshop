An overview of the internship and the development of the Smart Media Analytics project, broken down into weekly milestones with specific objectives:

- **[Week 1: AWS Basics]({{< ref "1.1-Week1.en.md" >}})**: Familiarized with the AWS ecosystem, practiced core services, and surveyed project directions.
- **[Week 2: Lambda & VPC]({{< ref "1.2-Week2.en.md" >}})**: Researched and configured Amazon VPC network infrastructure, Subnets, and AWS Lambda.
- **[Week 3: Project Orientation]({{< ref "1.3-Week3.en.md" >}})**: Onboarded at the company, learned internal workflows, and defined the project direction.
- **[Week 4: Backend & Docker]({{< ref "1.4-Week4.en.md" >}})**: Initialized the FastAPI Backend structure, practiced Docker packaging, and explored ECR/ECS Fargate.
- **[Week 5: Database & API]({{< ref "1.5-Week5.en.md" >}})**: Integrated PostgreSQL and Redis databases, and finalized the design of core management APIs.
- **[Week 6: AI Pipeline & Testing]({{< ref "1.6-Week6.en.md" >}})**: Deployed AI services via Docker, synchronized the development environment, and fully tested the Backend.
- **[Week 7: Vector Store & Media]({{< ref "1.7-Week7.en.md" >}})**: Completed the asynchronous AI pipeline (Scene, Image Caption, Transcription) and integrated ChromaDB.
- **[Week 8: Frontend Init]({{< ref "1.8-Week8.en.md" >}})**: Built the Frontend platform with React, managed centralized States, and made the first API connections.
- **[Week 9: Bedrock & WebSocket]({{< ref "1.9-Week9.en.md" >}})**: Connected AWS Bedrock, handled real-time uploads via WebSocket, and optimized Vector Embeddings.
- **[Week 10: Dashboard & Audio]({{< ref "1.10-Week10.en.md" >}})**: Built the Audio Waveform, finalized the interactive Dashboard, and adjusted AWS architecture based on feedback.
- **[Week 11: End-to-End & UI/UX]({{< ref "1.11-Week11.en.md" >}})**: Connected the E2E AI pipeline, packaged for Fargate, and overhauled the UI to an App-like standard.
- **[Week 12: CI/CD & Deployment]({{< ref "1.12-Week12.en.md" >}})**: Automated CI/CD, set up CloudWatch monitoring, and completed the comprehensive A-Z Workshop documentation.
