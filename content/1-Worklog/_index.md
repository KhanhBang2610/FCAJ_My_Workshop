Tổng quan quá trình thực tập và phát triển dự án nhóm trên AWS, được chia nhỏ thành các chặng đường theo từng tuần với mục tiêu cụ thể:

- **[Tuần 1: Kickoff & AWS Fundamentals]({{< ref "1.1-Week1.md" >}})**: Tham gia Kickoff FCAJ Bootcamp 2026, tìm hiểu quy định AWS, tạo tài khoản AWS và kết nối nhóm.
- **[Tuần 2: Explore AWS Tasks & Credit]({{< ref "1.2-Week2.md" >}})**: Hoàn thành 5 task Explore AWS (EC2, Bedrock, Budgets, Lambda, RDS) để nhận credit và học module 02.
- **[Tuần 3: Onboarding & Project Setup]({{< ref "1.3-Week3.md" >}})**: Làm quen môi trường công ty, phân công công việc Agile/Scrum và khởi tạo GitHub repo cho dự án nhóm.
- **[Tuần 4: Module Learning & Architecture]({{< ref "1.4-Week4.md" >}})**: Học các module AWS 04-05, nghiên cứu các bài AWS Blog và họp nhóm thiết kế kiến trúc.
- **[Tuần 5: AWS Architecture Design]({{< ref "1.5-Week5.md" >}})**: Thống nhất danh sách dịch vụ AWS sử dụng (Route 53, CloudFront, Cognito, Lambda, DynamoDB, SES, CI/CD...) và hoàn thiện sơ đồ kiến trúc.
- **[Tuần 6: Frontend Bootstrap]({{< ref "1.6-Week6.md" >}})**: Khởi tạo dự án React + Vite + TypeScript + Tailwind CSS, dựng các UI primitives và họp demo.
- **[Tuần 7: Authentication UI]({{< ref "1.7-Week7.md" >}})**: Phát triển giao diện Login, Register, Zod validation, cấu hình JWT interceptors và Protected Routes.
- **[Tuần 8: Admin Panel UI]({{< ref "1.8-Week8.md" >}})**: Xây dựng Admin Panel quản lý người dùng, phân quyền Role, Invite Member và Project Settings.
- **[Tuần 9: Kanban Task Board UI]({{< ref "1.9-Week9.md" >}})**: Triển khai bảng Kanban với 4 cột trạng thái (dnd-kit), tạo Modal CRUD Task và Assignee.
- **[Tuần 10: Activity Log & Realtime]({{< ref "1.10-Week10.md" >}})**: Xây dựng Activity Feed audit trail, tích hợp Socket thời gian thực và demo E2E.
- **[Tuần 11: E2E Integration Testing]({{< ref "1.11-Week11.md" >}})**: Kiểm thử tích hợp E2E toàn hệ thống, xử lý bug và merge code vào nhánh main.
- **[Tuần 12: Workshop Documentation]({{< ref "1.12-Week12.md" >}})**: Soạn thảo tài liệu Workshop hướng dẫn triển khai toàn bộ kiến trúc dự án.
